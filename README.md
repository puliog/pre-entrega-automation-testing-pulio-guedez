El propósito del proyecto

Validar los conocimientos adquiridos durante la primera etapa del curso, en una pre entrega que contiene las actividades, creando test para acciones basicas en una web ecommerce:
Login, Validar el titulo que se muestra en la URL, que la url sea la esperada, que se muestren productos, validar que se muestre el nombre y precio de al menos un producto.

Las tecnologías utilizadas

Selenium
Pytest
Git

Cómo instalar las dependencias

Selenium: pip install selenium
Pytest: pip install pytest
Generador de reportes: pip install pytest-html

Cómo ejecutar las pruebas

Todas las pruebas modo verbose
py -m pytest -v

Prueba por prueba: 
Ejemplo matcheo por pruebas
py -m pytest -k add_to_cart
